# 技能演进记录

## 演进规则

### 新技能生成
- 触发条件：某类问题现有技能连续2次精度 < 3
- 流程：分析失败原因 → 提炼新策略 → 编写技能卡片 → 入库 → 更新index

### 技能迭代
- 触发条件：某技能输出格式不够高效或覆盖不全
- 流程：提出改进方案 → 编写新版本 → 替换旧版 → 记录变更

### 技能淘汰
- 触发条件：某技能连续5次精度 < 3
- 流程：标记低效 → 尝试合并 → 若无法优化则移除 → 记录
