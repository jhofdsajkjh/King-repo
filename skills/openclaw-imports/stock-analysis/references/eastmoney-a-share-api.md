# A股数据提取速查

## 东方财富 API（主力）

### 实时行情
```bash
curl "http://push2.eastmoney.com/api/qt/stock/get?secid=0.{CODE}&fields=f43,f44,f45,f46,f47,f48,f57,f58,f60,f116,f117,f162,f163,f168,f170,f171,f174,f175"
```

**核心字段对照表**

| 字段 | 含义 | 示例 |
|------|------|------|
| f43 | 最新价（÷100） | 2251→22.51 |
| f44 | 最高价（÷100） | 2259→22.59 |
| f45 | 最低价（÷100） | 2160→21.60 |
| f46 | 今开（÷100） | 2246→22.46 |
| f47 | 成交量（÷10000=万股） | 128708→12.87万 |
| f48 | 成交额（÷1e8=亿） | 2.83e8→2.83亿 |
| f57 | 股票代码 | 002866 |
| f58 | 股票名称 | 传艺科技 |
| f60 | 昨收（÷100） | 2249→22.49 |
| f116 | 总市值（÷1e8=亿） | 65.17亿 |
| f117 | 流通市值（÷1e8=亿） | 40.79亿 |
| f162 | 市盈率(动)（÷100） | 10651→106.51 |
| f163 | 市盈率(静)（÷100） | 7592→75.92 |
| f168 | 换手率（÷100=%） | 712→7.12% |
| f170 | 涨跌幅（÷100=%） | 18→+0.18% |
| f171 | 量比 | 440→4.40 |
| f174 | 52周最高（÷100） | 3000→30.00 |
| f175 | 52周最低（÷100） | 1546→15.46 |
| f84 | 总股本（÷1e8=亿股） | 2.90亿 |
| f85 | 流通股本（÷1e8=亿股） | 1.81亿 |

**注意**：字段值通常需要÷100还原（价格/PE/换手率），成交量÷10000=万股，成交额÷1e8=亿。

### 涨跌幅计算
```python
change_pct = (price - yesterday) / yesterday * 100
change_amt = price - yesterday
```

## 股票代码规则

| 前缀 | 市场 |
|------|------|
| 600/601/603/688 | 上交所主板 |
| 000/001 | 深交所主板 |
| 002 | 中小板（深） |
| 300 | 创业板（深） |
| 688 | 科创板（上） |

secid 规则：`0.{代码}` 即可，0 代表沪深。

## 关键风险指标

- **量比 > 5**：异常放量，谨慎
- **PE < 0 或 亏损**：无 PE 支撑，业绩差
- **距52周高点 < 10%**：接近前高，可能遇阻
- **放量滞涨**：价格不跟量涨，疑似出货
