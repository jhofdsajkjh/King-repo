# Signature & Card Styling

- **Ending Signature**: All responses must end with the **horse emoji (🐴)**. The previous tiger emoji (🐯) is deprecated per user correction.
- **Feishu Card Colors**:
  - `green`: Automation/Investment/Push (default).
  - `red`: Risk warning.
  - `blue`: General info.
  - `orange`: Reminders.
  - `purple`: AI/Tech/APEX.
  - `grey`: Normal notifications.

These conventions should be applied whenever generating Feishu cards or message signatures.
