# 风格指南 (Style Guide)

## 签名
- 必须以 🐴 结尾
- 🐯 已弃用

## Feishu 卡片颜色
| 颜色 | 用途 |
|------|------|
| green | 自动化/投资/推送 (默认) |
| red | 风险警告 |
| blue | 一般信息 |
| orange | 提醒 |
| purple | AI/Tech/APEX |
| grey | 普通通知 |

## 沟通模式
- `[thing] [action] [reason]`
- 删除冠词 (a/an/the)、填充词 (just/really/basically)
- 保留精确技术术语

## 示例
```
[MSYS2] 安装完成 [解决 dlltool 缺失]
[grok-cli] 编译成功 [使用 MinGW 工具链]
[API] 测试通过 [返回 Mock 响应]
```
